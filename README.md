# AI 사주 운세 웹사이트

출생연도, 월, 일, 시간, 성별을 입력하면 한국어와 중국어로 사주팔자, 오행 분석, 대운 분석, 오늘의 운세, 재물운, 건강운, 직업운을 보여주는 초보자용 정적 웹사이트입니다.

## 실행 방법

초보자라면 아래 방법 중 하나를 선택하세요.

### 방법 1: 파일로 바로 열기

1. 이 폴더에서 `index.html` 파일을 찾습니다.
2. 파일을 더블클릭해 웹 브라우저로 엽니다.
3. 필요하면 상단의 **한국어 / 中文** 버튼으로 언어를 선택합니다.
4. 출생 정보를 입력하고 **운세 보기** 버튼을 누릅니다.
5. 결과가 표시되면 **결과 인쇄** 버튼으로 사주 결과만 깔끔하게 인쇄할 수 있습니다.

### 방법 2: 간단한 로컬 서버로 실행하기

Python이 설치되어 있다면 터미널에서 아래 명령어를 실행합니다.

```bash
python3 -m http.server 8000
```

그다음 브라우저에서 <http://localhost:8000> 주소를 엽니다.

## 파일 구성

- `index.html`: 화면 구조와 입력 폼
- `styles.css`: 디자인과 반응형 레이아웃
- `app.js`: 사주 계산, 오행 분석, 한국어/중국어 운세 문구 생성, 결과 인쇄 및 언어 전환 로직

## 참고

이 프로그램은 전통 명리학의 기본 규칙을 단순화해 재미용으로 제공하는 웹 앱입니다. 건강, 투자, 진로 등 중요한 결정은 반드시 전문가와 상담하세요.

## GitHub Pages 배포 방법

이 저장소는 별도 빌드가 필요 없는 정적 사이트이므로 GitHub Pages에 바로 배포할 수 있습니다.

### 1. GitHub에 저장소 올리기

```bash
git remote add origin https://github.com/사용자명/저장소명.git
git push -u origin work
```

이미 원격 저장소가 있다면 `git remote add`는 생략하고 현재 브랜치를 푸시하세요.

### 2. GitHub Pages 활성화

1. GitHub 저장소 페이지로 이동합니다.
2. **Settings → Pages** 메뉴를 엽니다.
3. **Build and deployment**의 **Source**를 **GitHub Actions**로 선택합니다.
4. 현재 브랜치에 포함된 `.github/workflows/deploy-pages.yml` 워크플로가 자동으로 사이트를 배포합니다.

### 3. 배포 주소 확인

배포가 끝나면 GitHub 저장소의 **Settings → Pages** 화면 또는 Actions 로그에서 아래 형식의 주소를 확인할 수 있습니다.

```text
https://사용자명.github.io/저장소명/
```

사용자 사이트 저장소(`사용자명.github.io`)에 배포하는 경우 주소는 아래 형식입니다.

```text
https://사용자명.github.io/
```
